色の指定(その２)

(update:2025/3/30)


概 要

機　能

Gdk::RGBAには様々な色の指定方法が整備されています。ここではいくつかの色の指定方法についてご紹介します。

使い方

  1. Gdk::RGBA 変数名( Glib::ustring & value )

    2. 変数の宣言時に変数名と合わせて色を指定します。

    例. Gdk::RGBA m_color( "green" ), Gdk::RGBA m_color( "#00ff00" )

  2. set_rgba

    3. 赤、緑、青の3色で色を表します。実数で表現します。

    void Gdk::RGBA::set_rgba( float red, // 赤 0.0～1.0
    float green, // 緑 0.0～1.0
    float blue, // 青 0.0～1.0
    float alpha=1.0 ) // ｱﾙﾌｧ要素

  3. set_rgba_u

    4. 赤、緑、青の3色で色を表します。正数で表現します。

    void Gdk::RGBA::set_rgba_u( gushort red, // 赤 0～65535
    gushort green, // 緑 0～65535
    gushort blue, // 青 0～65535
    gushort alpha=65535 ) // ｱﾙﾌｧ要素 0～65535

  4. hsv

    5. 色相、彩度、明度の三つの要素で色を表します。

    void Gdk::RGBA::set_hsv( float h, // 色相 0.0～360.0
    float s, // 彩度 0.0～1.0
    float v, // 明度 0.0～1.0
    float alpha=1.0 ) // ｱﾙﾌｧ要素

  5. hsl

    6. 色相、彩度、輝度の三つの要素で色を表します。

    void Gdk::RGBA::set_hsl( float h, // 色相 0.0～360.0
    float s, // 彩度 0.0～1.0
    float l, // 輝度 0.0～1.0
    float alpha=1.0 ) // ｱﾙﾌｧ要素


サンプル プログラム


#include <gtkmm.h>

class Drawing : public Gtk::DrawingArea {
public:
    Drawing();
    virtual ~Drawing() = default;
protected:
    void on_draw(const Cairo::RefPtr<Cairo::Context>& cr, int width, int height);
};
  
Drawing::Drawing()
{
    set_draw_func( sigc::mem_fun( *this, &Drawing::on_draw));
}
  
void Drawing::on_draw(const Cairo::RefPtr<Cairo::Context>& cr, int width, int height) {  
   // background
    Gdk::Cairo::set_source_rgba( cr, Gdk::RGBA( "black" ));
    cr->paint();
   // Example
    // Gdk::RGBA 変数名(lightgreen)
    Gdk::RGBA m_color_x( "#90ee90" );
    cr->rectangle( 10.0, 5.0, width -20.0, 30.0 );
    Gdk::Cairo::set_source_rgba( cr, m_color_x );
    cr->fill();
      
    Gdk::RGBA m_color;
    cr->rectangle( 10.0, 45.0, width -20.0, 30.0 );
    // set_rgba(lightseagreen)
    m_color.set_rgba( 0.125, 0.698, 0.667, 0.9 );
    Gdk::Cairo::set_source_rgba( cr, m_color );
    cr->fill();
    // set_rgba_u(yellowgreen)
    cr->rectangle( 10.0, 85.0, width -20.0, 30.0 );
    m_color.set_rgba_u( 39578, 52685, 12850, 65535 );
    Gdk::Cairo::set_source_rgba( cr, m_color );
    cr->fill();
    // hsv(springgreen)
    cr->rectangle( 10.0, 125.0, width -20.0, 30.0 );
    m_color.set_hsv( 150.0, 100.0, 100.0, 1.0 );
    Gdk::Cairo::set_source_rgba( cr, m_color );
    cr->fill();
    // hsl(palegreen)
    cr->rectangle( 10.0, 165.0, width -20.0, 30.0 );
    m_color.set_hsl( 120, 0.93, 0.79, 1.0 );
    Gdk::Cairo::set_source_rgba( cr, m_color );
    cr->fill();
}
  
class MyWindow : public Gtk::Window
{
public:
    MyWindow();
protected:
    Drawing my_draw;  
};
  
MyWindow::MyWindow()
{
    set_title( "Gdk::RGBA No.2" );
    set_default_size( 320, 240 );
    set_child( my_draw );
}
  
int main(int argc, char* argv[]) {
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
