Labelの中の文字列の装飾(色)

(update:2025/4/29)


概 要

機　能

Labelの中の文字列の色を変更します。

使い方

  1. gtkmm4では、Pango Markup syntaxを用いてwidgetの中のテキスト(文字列)に装飾を加えることができます。
  2. The <span> Attributes # を使用してフォントの色、背景色を指定することができます。
  3. 具体的には、set_markup()の引数として、表示するテキスト(文字列)を<span>~</span>で囲みます。
  4. <span>~</span>で囲まれた範囲の書式が変更されます。
The <span> Attributes
書式 意味 記載例
foreground(fgcolor・color) 文字の色 e.g. RGB : #00FF00 RGBA : #00FF007F
background(bgcolor) 文字の背景色 e.g. RGB : #00FF00 RGBA : #00FF007F
alpha(fgalpha) アルファ値 e.g. 1～65536 50%
background_alpha(bgalpha) 背景色のアルファ値 e.g. 1～65536 50%

サンプル プログラム


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
  
private:
    Gtk::Box m_vbox;
    Gtk::Label m_label1, m_label2, m_label3, m_label4, m_label5;
};
  
MyWindow::MyWindow()
 : m_vbox( Gtk::Orientation::VERTICAL )
{
    set_title("Label(pango markup - Text Attributes and Markup)");
    m_vbox.set_margin( 15 );
    m_vbox.set_spacing( 10 );
    m_vbox.set_expand();
    set_child( m_vbox );
   // fgcolor
    m_label1.set_text( "<span fgcolor='#ffff00' bgcolor='#0000cd'>>Fortune comes in the merry gate.</span>" );
    m_label1.set_halign( Gtk::Align::CENTER );
    m_label1.set_use_markup( true );
    m_vbox.append( m_label1 );
   // bgcolor
    m_label2.set_markup( "<span fgcolor='#0000cd' bgcolor='#ffff00'>Fortune comes in the merry gate.</span>" );
    m_label2.set_halign( Gtk::Align::CENTER );
    m_vbox.append( m_label2 ); 
   // fgalpha
    m_label3.set_markup( "<span fgcolor='#0000cd' fgalpha='50%'>Fortune comes in the merry gate.</span>" );
    m_label3.set_halign( Gtk::Align::CENTER );
    m_vbox.append( m_label3 ); 
   // bgalpha
    m_label4.set_markup( "<span bgcolor='#0000cd' bgalpha='50%'>Fortune comes in the merry gate.</span>" );
    m_label4.set_halign( Gtk::Align::CENTER );
    m_vbox.append( m_label4 ); 
}
  
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.example" );   
    return app->make_window_and_run<MyWindow>(argc, argv);
}
