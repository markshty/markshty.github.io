文字列のレイアウト

(update:2025/4/19)


概 要

機　能

グラフィック画面(Gtk::DrawingArea)に文字列を表示する際のレイアウトを設定します。

使い方

  1. 幅の設定

    2. 文字列の描画領域の幅を設定します。Pango::Layoutでは、設定する数値を1/Pango::SCALEピクセル単位で指定します。

    例．文字列を表示する幅を300ピクセルとする場合 : 300 * Pango::SCALE

    【関　数】

    void Pango::Layout::set_width( int width )

    【返　値】 : 無し


  2. 高さの設定

    3. 文字列の描画領域の高さを設定します。

    例．文字列を表示する高さを240ピクセルとする場合 : 240 * Pango::SCALE

    【関　数】

    void Pango::Layout::set_width( int width )

    【返　値】 : 無し


  3. インデントの設定

    4. 文字列の先頭のインデントの量を設定します。

    例．文字のサイズを14ピクセルとする : 14 * Pango::SCALE

    【関　数】

    void Pango::Layout::set_indent( int indent )

    【返　値】 : 無し


  4. 行間の設定

    5. 文字列を表示する行が複数に至る場合の行間を文字列の先頭の段下げを量を設定します。factorのデフォルトは0.0です。

    次行の高さをheight2とした場合に、factorが0でなければ基準線２＝基準線１ + factor * height2となります。factorのデフォルト値は0.0です。1とすると行間が０になります。2にすると1行あいたようになります。サンプルプログラムでは、fctorを1.5とすることにより、行間を0.5行分空けています。

    【関　数】

    void Pango::Layout::set_line_spacing( float factor )

    【返　値】 : 無し


  5. 文字列の幅寄せの設定

    6. 文字列の幅寄せを設定します。

    【関　数】

    void Pango::Layout::set_alignment( Alignment alignment )

    【返　値】 : 無し


    Pango::Alignment
    書式 内容
    LEFT 右寄せ
    CENTER 中央
    RIGHT 左寄せ


  6. 文字列の均等割り付け

    7. 文字列を行一杯に均等割り付けを行うかどうかを指定します。

    【関　数】

    void Pango::Layout::set_justify( bool justify=true )

    【返　値】 : 無し


  7. 文字列の設定

    8. マークアップされた文字列を設定します。

    Pangommでは、Pango Markup syntaxを用いてテキスト(文字列)に装飾を加えることができます。

    【関　数】

    void Pango::Layout::set_markup( const Glib::ustring & markup )

    【返　値】 : 無し



サンプル プログラム


#include <gtkmm.h>

const Glib::ustring text =
      "山路を登りながらこう考えた。\n"
      "<span underline_color='#0000ff'><u>智に働けば角が立つ。"
      "情に棹させば流される。意地を通せば窮屈だ。"
      "とかくに人の世は住みにくい。</u></span>\n"
      "住みにくさが高じると、安い所へ引き越したくなる。"
      "どこへ越しても住みにくいと悟った時、詩が生まれて画ができる。\n"
      "人の世を作ったものは神でもなければ鬼でもない。"
      "やはり向う三軒両隣にちらちらするただの人である。"
      "ただの人が作った人の世が住みにくいからとて、越す国はあるまい。"
      "あれば人でなしの国へ行くばかりだ。"
      "人でなしの国は人の世よりもなお住みにくかろう。\n"
      "（<i>夏目漱石</i>・「<b>草枕</b>」より）\n";
  
class Drawing : public Gtk::DrawingArea {
public:
    Drawing();
    virtual ~Drawing() = default;
protected:
    void on_draw(const Cairo::RefPtr<Cairo::Context>& cr, int width, int height);
};
  
Drawing::Drawing()
{
    set_draw_func( sigc::mem_fun( *this, &Drawing::on_draw));
}
  
void Drawing::on_draw(const Cairo::RefPtr<Cairo::Context>& cr, int width, int height) {
     
    Glib::RefPtr<Pango::Layout> layout;
    Pango::FontDescription font;
    const int font_size = 14;
    const int margin = 10;
  
    cr->set_source_rgba( 1.0, 1.0, 1.0, 1.0 );
    cr->paint();
  
    layout = create_pango_layout( "" );
    layout->set_width(  Pango::SCALE * ( width  - margin * 2 ));
    layout->set_height( Pango::SCALE * ( height - margin * 2 ));
    layout->set_indent( Pango::SCALE * font_size );
    layout->set_line_spacing( 1.5 );
    layout->set_alignment( Pango::Alignment::LEFT );
    layout->set_justify( TRUE );
    layout->set_markup( text );
  
    font.set_family( "Sans Serif" );
    font.set_size( font_size * Pango::SCALE );
    layout->set_font_description( font );
  
    cr->move_to( margin, margin );
    cr->set_source_rgba( 0.0, 0.0, 0.0, 1.0 );
    layout->show_in_cairo_context( cr );
}
  
class MyWindow : public Gtk::Window
{
public:
    MyWindow();
protected:
    Drawing my_draw;
};
  
MyWindow::MyWindow()
{
    set_title( "Pango layout Sample" );
    set_default_size( 545, 380 );
    set_child( my_draw );
}
  
int main(int argc, char* argv[]) {
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
