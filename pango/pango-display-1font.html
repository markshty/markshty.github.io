フォントの指定

(update:2025/4/29)


概 要

機　能

グラフィック画面(Gtk::DrawingArea)に文字列を表示する際、フォントの種類、大きさなどを指定します。

使い方

  1. フォントの種類

    2. フォントの種類を指定します。

    自分のPCの環境で使用できるフォントを調べるためには、debianなどではterminalで

    % fc-list | cut -d : -f 2 | cut -d , -f 1 | less

    を実行すると、自分のPCの環境にインストールされているフォントの一覧が表示されます。

    【関　数】

    void Pango::Fontdescription::set_family( const Glib::ustring & family )

    【返　値】 : 無し


  2. フォントの太さ(重み)

    3. 重みは、100～1000の数値で定義されています。

    Pango::Weight
    書式 内容
    THIN range : 100
    ULTRALIGHT range : 200
    LIGHT range : 300
    SEMILIGHT range : 350
    BOOK range : 380
    NORMAL range : 400
    MEDIUM range : 500
    SEMIBOLD range : 600
    BOLD range : 700
    ULTRABOLD range : 800
    HEAVY range : 900
    ULTRAHEAVY range : 1000

    【関　数】

    void Pango::Fontdescription::set_weight( Weight weight )

    【返　値】 : 無し


  3. フォントのスタイル(傾斜)

    4. フォントの傾斜のスタイルを指定します。

    Pango::Style
    書式 内容
    NORMAL 直立
    ITALIC フォントはイタリック体で傾斜
    OBLIQUE フォントはロマン体で傾斜

    【関　数】

    void Pango::Fontdescription::set_style( Style style )

    【返　値】 : 無し


  4. フォントのサイズ

    5. フォントのサイズを指定します。10ポイントで表示したい場合は、10 × Pango::SCALE の値を引数として指定します。

    DTPソフトでは、1ポイント＝1/72インチと定義されることが一般的です。DPI(dot/inch)が96の場合、10ポイント×96/72⇒13.3ピクセルとなり、10ポイントの文字が13.3ピクセルとして表示されます。

    【関　数】

    void Pango::Fontdescription::set_size( int size )

    【返　値】 : 無し



サンプル プログラム


#include <gtkmm.h>
  
class Drawing : public Gtk::DrawingArea {
public:
    Drawing();
    virtual ~Drawing() = default;
protected:
    void on_draw(const Cairo::RefPtr<Cairo::Context>& cr, int width, int height);
};
  
Drawing::Drawing()
{
    set_draw_func( sigc::mem_fun( *this, &Drawing::on_draw));
}
  
void Drawing::on_draw(const Cairo::RefPtr<Cairo::Context>& cr, int width, int height) {
    Pango::FontDescription font;
    Glib::RefPtr<Pango::Layout> layout;
    int text_width;
    int text_height;
  
    font.set_family( "VL Pゴシック" );
    font.set_size( 20 * Pango::SCALE );
  
    layout = create_pango_layout( "" );
    layout->set_text( "Good morning!" );
    layout->set_font_description( font );
  
    layout->get_pixel_size( text_width, text_height );
    cr->move_to(( width - text_width )/2, ( height / 3.0 - text_height ) / 2.0 );
   // Normal
    layout->show_in_cairo_context( cr );

   // Weight::HEAVY
    font.set_weight( Pango::Weight::HEAVY );
    font.set_style( Pango::Style::NORMAL );
    layout->set_font_description( font );
    cr->rel_move_to(  0.0, text_height * 2.0 );
    layout->show_in_cairo_context( cr );

   // Style::ITALIC
    font.set_weight( Pango::Weight::NORMAL );
    font.set_style( Pango::Style::ITALIC );
    layout->set_font_description( font );
    cr->rel_move_to(  0.0, text_height * 2.0 );
    layout->show_in_cairo_context( cr );
}
  
class MyWindow : public Gtk::Window
{
public:
    MyWindow();
protected:
    Drawing my_draw;
};
  
MyWindow::MyWindow()
{
    set_title( "Pango FontDescription" );
    set_default_size( 320, 240 );
    set_child( my_draw );
}  
  
int main(int argc, char* argv[]) {
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
