CSS 放射形グラデーション

(update:2025/7/25)

概 要

機　能

放射形グラデーションは、ウィジットの中心から放射状に広がるグラデーションです。2色以上の色が連続的に変化することでWindowの背景やウィジットに様々な色彩の効果を加えることができます。


使い方

  1. 形状

    2. 放射形グラデーションの形状は、正円(circle)と楕円(ellipse)の2種類です。デフォルトは正円です。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
/* 円 */
#m-label1 {
    background-image: radial-gradient( circle, yellow 0%, blue 100%);
}

/* 楕円 */
#m-label2 {
    background-image: radial-gradient( ellipse 60% 40%, yellow 0%, blue 100%);
}
    実行結果


  2. サイズ

    3. 放射形グラデーションでは、グラデーションのサイズを指定することができます。

    書　式 内　容
    closest-side 円の中心から領域の最も近い辺に内接
    farthest-side 円の中心から領域の最も遠い辺に内接
    closest-corner 円の中心から領域の最も近い頂点に接する
    farthest-corner 円の中心から領域の最も遠い頂点に接する
    任意の数値＋単位 水平・垂直方向の半径を半角スペースで区切って、単位付きの数値で指定
    ％値 水平・垂直方向の半径を半角スペースで区切って、％値で指定。値は、ウィジットの幅と高さに対する割合

    ※円は、数値でのみ半径を指定することができます。

    ※楕円のみの場合は、長さまたは％値を使って楕円のサイズを指定することができます。


    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test2.cpp
    style.css
    
/* 楕円 closest-side */
#m-label1 {
    background-image: radial-gradient(
        ellipse closest-side,
        tomato,
        gold    10%,
        #00fa9a 50%,
        Bisque 
    );
}

/* 楕円 fathest-corner */ 
#m-label2 {
    background-image: radial-gradient(
        ellipse farthest-corner at 90% 90%,
        tomato,
        gold    10%,
        #00fa9a 50%,
        Bisque
    );
}

/* 円 closest-side */
#m-label3 {
    background-image: radial-gradient(
        circle closest-side at 25% 75%,
        tomato,
        gold    10%,
        #00fa9a 50%,
        Bisque 
    );
}

/* 楕円 半径を％値または長さ */
#m-label4 {
    background-image: radial-gradient(
        ellipse 50% 50px,    /* 水平方向の半径 垂直飛行の半径 */
        tomato,
        gold    10%,
        #00fa9a 50%,
        Bisque
    );
}

/* 円 半径を実数 */
#m-label5 {
    background-image: radial-gradient(
        circle 50px,         /* 半径 */
        tomato,
        gold    10%,
        #00fa9a 50%,
        Bisque
    );
}
    実行結果


  3. 中心の位置

    4. グラデーションの中心位置を指定します。デフォルトは at center となります。

    書　式 内　容
    at top 領域の上辺が中心となる
    at top right 領域の右上角が中心となる
    at right 領域の右辺が中心となる。
    at bottom right 領域の右下角が中心となる。
    at bottom 領域の下辺が中心となる。
    at bottom left 領域の左下角が中心となる。
    at left 領域の左辺が中心となる。
    at top left 領域の左上角が中心となる。
    at center 領域の中央が中心となる。
    任意の数値＋単位 中心の座標を単位付きの数値で指定。基準は領域の左上角。
    ％値 中心の座標を％値で指定。値は、領域の幅と高さに対する割合

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test3.cpp
    style.css
    
window {
    background-image: radial-gradient(
        circle at 0% 50%,
        tomato    10px,
        gold      30%,
        #00fa9a   50%,
        Bisque
    );
}
    実行結果


  4. 色経由点(color stops)の配置

    5. color stopsの配置は、任意の位置を指定することができます。指定する値としては、パーセント値を使用するか、放射・線形グラデーションの場合は、具体的な長さを指定します。

    color stopsをパーセント値で指定する場合は、0%が開始点を表し、100％が終了点を表します。color stopsを指定しない場合は、その値はシステムによって自動的に計算され、最初のcolor stopsは 0% となり、最後のcolor stopsは、 100% となり、他のcolor stopsは、両側のcolor stopsの中間の位置となります。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test3.cpp
    style.css
    
#m-label {
    background-image: radial-gradient(
        circle 110px,
        tomato,
        gold    10%,
        #00fa9a 50%,
        Bisque 
    );
}
    実行結果


サンプル プログラム１


#include <gtkmm.h>
#include <iostream>
#include <gtkmm/cssprovider.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;

private:
    Gtk::Box   m_box;
    Gtk::Label m_label1, m_label2;

protected:
   // signal handler:
    static void on_parsing_error( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err );
   // child widgets:
    Glib::RefPtr<Gtk::CssProvider> m_refCssProvider;
};

MyWindow::MyWindow()
{
    set_title( "Css radial-gradient" );
    set_child( m_box );
    m_box.set_margin( 20 );
    m_box.set_spacing( 20 );

    m_label1.set_size_request( 150, 150 );
    m_label1.set_name( "m-label1" );
    m_box.append( m_label1 );

    m_label2.set_size_request( 150, 150 );
    m_label2.set_name( "m-label2" );
    m_box.append( m_label2 );

   // Load extra CSS file
    m_refCssProvider = Gtk::CssProvider::create();   
    Gtk::StyleContext::add_provider_for_display( get_display(), m_refCssProvider,
        GTK_STYLE_PROVIDER_PRIORITY_APPLICATION );
    m_refCssProvider->signal_parsing_error().connect(
        []( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err )
        { on_parsing_error( sec, err ); }
    );   
    m_refCssProvider->load_from_path( "style.css" );
}

void MyWindow::on_parsing_error( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err )
{
    Gtk::CssLocation s_location, e_location;

    std::cerr << "on_parsing_error(): " << err.what() << std::endl;
    if ( sec ) {
        Glib::RefPtr<const Gio::File> file = sec->get_file();
        if ( file ) {
            std::cerr << "  URI = " << file->get_uri() << std::endl;
        }

        s_location = sec->get_start_location();
        e_location = sec->get_end_location();
        std::cerr << "  start : " << s_location.get_lines()+1
                  << ", end   : " << e_location.get_lines()+1 << std::endl;
        std::cerr << "  s_pos : " << s_location.get_line_chars()
                  << ", e_pos : " << e_location.get_line_chars() << std::endl;
    }
}

int main( int argc, char* argv[] )
{
    auto app = Gtk::Application::create( "gtkmm4.example" );   
    return app->make_window_and_run<MyWindow>( argc, argv );
}
