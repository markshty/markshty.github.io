Labelの中の文字列の位置揃え


概 要

機　能

labelの中の文字列の位置揃えを行います。

使い方

  • set_justify(引数)を実行します。
  • 引数は次の4通りです。
    • 左寄せ(Gtk::Justification::LEFT)
    • 中央(Gtk::Justification::CENTER)
    • 右寄せ(Gtk::Justification::RIGHT)
    • 均等(Gtk::Justification::FILL)
  • デフォルトは、中央です。
  • 文字列の中に"\n"を挟むと改行します。

サンプル プログラム

update:2024/11/8


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;  
private:
    Gtk::Box m_vbox;
    Gtk::Label m_label1, m_label2, m_label3, m_label4;
};
  
MyWindow::MyWindow()
 : m_vbox( Gtk::Orientation::VERTICAL ),
   m_label1( "The marathon will be\n held tomorrow" ),
   m_label2( "The marathon will be\n held tomorrow" ),
   m_label3( "The marathon will be\n held tomorrow" ),
   m_label4( "The marathon will be\n held tomorrow" )
{
     set_title("Label( justification )");
     m_vbox.set_margin( 15 );
     m_vbox.set_spacing( 10 );
     m_vbox.set_expand();
     set_child( m_vbox );
  
   // Left
     m_label1.set_justify( Gtk::Justification::LEFT );
     m_vbox.append( m_label1 ); 
   // Right
     m_label2.set_justify( Gtk::Justification::RIGHT );
     m_vbox.append( m_label2 ); 
   // Center
     m_label3.set_justify( Gtk::Justification::CENTER );
     m_vbox.append( m_label3 );
   // Fill
     m_label4.set_justify( Gtk::Justification::FILL );
     m_vbox.append( m_label4 );
}
  
int main(int argc, char* argv[])
{
      auto app = Gtk::Application::create( "gtkmm4.example" );   
      return app->make_window_and_run<MyWindow>(argc, argv);
}
