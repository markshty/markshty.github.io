画像データの回転

(update:2025/2/1)


概 要

機　能

画像データを回転させます。

使い方

  • Gdk::pixbufにセットされている画像データを90度単位で回転させます。
  • 画像データを回転させるためには、Gdk::Pixbuf::rotate_simple( Rotation angle )を使用します。
Rotation angle
書式 意味
Gdk::Pixbuf::Rotation::NONE 回転しない
Gdk::Pixbuf::Rotation::COUNTERCLOCKWISE ９０度回転
Gdk::Pixbuf::Rotation::UPSIDEDOWN １８０度回転
Gdk::Pixbuf::Rotation::CLOCKWISE ２７０度回転

サンプル プログラム


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
  
private:
    Gtk::Image m_image;
};
  
MyWindow::MyWindow()
{
    set_title( "画像データの回転" );
    set_default_size( 320, 240 );
  
    Glib::RefPtr<Gdk::Pixbuf> pixbuf = Gdk::Pixbuf::create_from_file( "./test.bmp" );
   // 画像データを回転(90度単位)
    pixbuf = pixbuf->rotate_simple( Gdk::Pixbuf::Rotation::COUNTERCLOCKWISE );
    m_image.set( pixbuf ); 
    set_child( m_image );
}

int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
