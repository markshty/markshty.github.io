Gtk::frame

(update:2025/7/25)

概 要

機　能

frameを表示します。

使い方

  1. frameのラベルのセット

    2. frameの上端にラベルをセットします。ラベルの配置は、デフォルトでは、フレームの上端の左端となります。

    【関　数】

    void Gtk::Frame::set_label( const Glib::ustring & label )

    【返　値】 : 無し


  2. frameの配置

    3. frameのラベルの配置をセットします。

    【関　数】

    void Gtk::Frame::set_label_align( Align xalign )

    【返　値】 : 無し


    Gtk::Align
    書式 内容
    FILL 枠のスペースの幅一杯に配置
    STRAT 左端あるいは上端に寄せる
    END 右端あるいは下端に寄せる
    CENTER 枠の中心に配置


  3. marginの指定

    4. frameの周囲のmarginをセットします。もし、marginが0ならば、frameの枠線は表示されません。

    【関　数】

    void Gtk::Widget::set_margin( int margin )

    【返　値】 : 無し


  4. frameに表示するwidgetをセット

    5. frameに子Widgetをセットします。Gtk::frameは、Single-item Containerですので単一のWidgetしか格納できません。複数のWidgetを格納する場合は、Gtk::BoxなどのMulti-item ContainerにWidgetを格納してから、Gtk::frameに格納します。

    【関　数】

    void Gtk::Frame::set_child( Widget & child )

    【返　値】 : 無し


  5. frameを親のwidgetにセット

    6. frameを子Widgetとして親Widgetにセットします。

    【関　数】

    void Gtk::Frame::set_child( Widget & child )

    【返　値】 : 無し


サンプル プログラム


#include <gtkmm.h>

class MyWindow : public Gtk::Window {
private:
    Gtk::Frame m_frame;
    Gtk::Image m_image;
public:
    MyWindow();
    virtual ~MyWindow() = default;
};
  
MyWindow::MyWindow()
{
    set_title( "Frame" );
    set_default_size( 320, 330 );
    m_image.set_expand( true );
   // 1.frameのラベルのセット
    m_frame.set_label( "< tree >" );
   // 2.frameの配置
    m_frame.set_label_align( Gtk::Align::CENTER );
   // 3.marginの指定
    m_frame.set_margin( 10 );
      
    m_image.set( "tree0.png" );
   // 4.frameに表示するwidgetをセット
    m_frame.set_child( m_image );
   // 5.frameを親のwidgetにセット
    set_child( m_frame );
} 
  
int main(int argc, char *argv[]) 
{
    auto app = Gtk::Application::create( "gtkmm4.examples" );
    return app->make_window_and_run<MyWindow>(argc, argv);
}
