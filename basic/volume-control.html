Volume Control

(update:2026/1/2)

概 要

目　的

Gtk::Scaleを使って、Volume Controlを作成します（gtk4では、Gtk::VolumeButtonの使用は推奨されていません。gtk5で削除される予定）


構　成

Volume Controlのプログラムの構成は次のとおりです。
  1. クラスの宣言
    • Gtk::Boxの派生クラスとして、Volume Controlクラスを宣言します。
    • Volume Controlクラスの中で変数として、Gtk::ImageとGtk::Scaleを宣言します。
    • Gtk::Imageはアイコンを格納し、Gtk::Scaleは、スライダーとなります。

  2. Volume Controlの実体の生成

    3. MyWindowの中でVolumu Controlクラスの実態を生成します。


  3. Volume Controlの初期状態の設定

    4. Volumu Controlのコンストラクタの中で、

    • スライダーの最小値、最大値、初期値、値の表示の有無、値の小数点以下の表示桁数、値の表示位置を設定します。
    • アイコンの初期値を設定します。
    • Gtk::ImageとGtk::ScaleをVolume Contorolにappendします。
    • スライダーが操作されたときのイベントの起動を設定します。

  4. イベントシグナルの設定

    5. スライダーを操作したときにイベントシグナル(signal_value_changed)を発生させてシグナルハンドラーon_scale_changed()を起動します。


  5. シグナルハンドラーの起動
    • スライダーの値の取得

      • スライダーが操作されたときに、スライダーの値をget_value()を実行して取得します。

    • アイコンの設定の変更

      • スライダーの値に応じて、set_from_icon_name()を実行し、アイコンの表示内容を変更します。



サンプル プログラム


#include <gtkmm.h>
#include <iostream>

// 1.クラスの宣言
class VolumeControl : public Gtk::Box {
public:
    VolumeControl();
    virtual ~VolumeControl() = default;
private:
    Gtk::Scale scale;
    Gtk::Image icon;
    void on_volume_changed();
};

// 3.Volume Controlの初期状態の設定
VolumeControl::VolumeControl()
 : Gtk::Box(Gtk::Orientation::HORIZONTAL)
{
    // 3.1 スライダー
    scale.set_orientation( Gtk::Orientation::HORIZONTAL );
    scale.set_range( 0.0, 1.0 );
    scale.set_value( 0.5 );
    scale.set_draw_value( true );
    scale.set_digits( 2 );
    scale.set_value_pos( Gtk::PositionType::RIGHT );
    scale.set_hexpand( true );
    // 3.2 アイコン
    icon.set_from_icon_name( "audio-volume-medium-symbolic" );
    // 3.3 Volume Controlへのappend
    append( icon );
    append( scale );
    // 4.イベントシグナルの設定
    scale.signal_value_changed().connect(sigc::mem_fun(*this, &VolumeControl::on_volume_changed));
}

// 5.シグナルハンドラーの起動
void VolumeControl::on_volume_changed()
{
    // 5.1 スライダーの値の取得
    double value = scale.get_value();
    std::cout << "Volume: " << value << std::endl;

    // 5.2 アイコンの設定の変更
    if ( value == 0.0 )
        icon.set_from_icon_name( "audio-volume-muted-symbolic" );
    else if ( value < 0.5 )
        icon.set_from_icon_name( "audio-volume-low-symbolic" );
    else if ( value < 0.8 )
        icon.set_from_icon_name( "audio-volume-medium-symbolic" );
    else
        icon.set_from_icon_name( "audio-volume-high-symbolic" );
}

class MyWindow : public Gtk::Window {
public:
    MyWindow();
    virtual ~MyWindow() = default;
private:
    // 2.Volume Controlの実体の生成
    VolumeControl volume_control;
};

MyWindow::MyWindow()
{
    set_title( "Volume Control" );
    set_default_size( 300, 50 );
    volume_control.set_margin( 10 );
    volume_control.set_spacing( 0 );
    set_child( volume_control );
}

int main( int argc, char* argv[] )
{
    auto app = Gtk::Application::create( "scale.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
