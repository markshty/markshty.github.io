CheckButton

(update:2025/12/14)

概 要

機　能

Gtk::CheckButtonは、クリックするたびにオンとオフといった2つの状態を切り替えるボタンです。


使い方

Gtk::CheckButtonウイジットの使い方は次のとおりです。
  1. ボタンの作成

    2. Gtk::CheckButtonウイジットの作成は、Gtk::CheckButtonのコンストラクタでラベルとなる文字列を指定します。あるいは、Gtk::CheckButtonのコンストラクタで定義しておいて、ラベルの文字列は、set_label( "文字列" ) を使って後から指定します。


  2. シグナルの発生

    3. ボタンをクリックしたときにイベントシグナル(signal_toggled)を発生させてシグナルハンドラーon_button_toggled()を起動します。


  3. get_active()

    4. 現在の状態を取得（ON/OFF）。


  4. set_active( bool )

    5. プログラム上でボタンのON(true)、OFF(false)を指定する。



サンプル プログラム


#include <gtkmm/application.h>
#include <gtkmm/window.h>
#include <gtkmm/checkbutton.h>
#include <iostream>

class MyWindow : public Gtk::Window
{
public:
  MyWindow();
  virtual ~MyWindow() = default;

protected:
  //Signal handlers:
  void on_button_toggled();
  //Child widgets:
  Gtk::CheckButton m_button;
};

MyWindow::MyWindow()
{
    set_title( "CheckButton" );

    m_button.set_label( "Check" );
    m_button.set_active( false );

    m_button.signal_toggled().connect( sigc::mem_fun( *this,
            &MyWindow::on_button_toggled ));

    m_button.set_margin( 10 );
    set_child( m_button );
}

void MyWindow::on_button_toggled()
{
    if( m_button.get_active() ) {
        std::cout << "チェックされています" << std::endl;
    }
    else {
        std::cout << "チェックが外されました" << std::endl;
    }
}

int main( int argc, char *argv[] )
{
    auto app = Gtk::Application::create( "check.test" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
