SpinButton

(update:2025/12/20)

概 要

機　能

Gtk::SpinButtonは、数値を入力したり選択するためのウィジェットです。ユーザーは直接数値を入力するか、上下の矢印ボタンで値を増減することによって数値を設定することができます


使い方

Gtk::SpinButtonの使い方は次のとおりです。
  1. set_range( double min, double max )

    2. Gtk::SpinButtonに入力できる最小値と最大値の範囲を設定するメソッドです。引数は(min,max)の二つの数値です。ユーザーが入力できる値をこの範囲に制限します。範囲外の数値を入力すると、自動的に最小値または最大値に補正されます。


  2. set_increments( double step, double page)

    3. Gtk::SpinButtonの値を増減させるときの刻み幅を設定するメソッドです。第1引数が通常のステップ幅、第2引数がページステップ幅です。

    • 第１引数stepは、＋－ボタンをクリックしたり上下矢印キーを押したときに増減する量です。
    • 第２引数pageは、PageUp / PageDown キーを押したときに増減する量です。
    • 値の範囲は set_range() で制御されるため、範囲外にはなりません。

  3. set_value( double val )

    4. Gtk::SpinButtonに値を直接設定する メソッドです。

    • 引数valに数値を渡すと、その値が直ちに表示され、内部状態も更新されます。
    • 範囲がset_range(min, max)によって設定されている場合、範囲外の値を渡すと自動的に補正されます。
    • イベント連動として、signal_value_changed()が接続されている場合、set_value()による変更でもシグナルが発生します。

  4. get_value()

    5. Gtk::SpinButtonに現在設定されている数値を取得するメソッドです。

    • 戻り値は double型で、SpinButtonの内部値を返します。
    • 現在値の取得として、ユーザーが入力した値、またはプログラムで設定した値を返します。
    • - 小数点桁数: set_digits() の設定に従って表示は丸められますが、get_value() は内部の実際の値を返します。
    • set_range()で範囲を指定している場合は、範囲外の値は補正されてから返されます。
    • set_digits()の設定に従って、表示は丸められますが、戻り値の小数点桁数は、get_value()は、内部の実際の値を返します。

  5. set_digits( int val )

    6. Gtk::SpinButtonに表示される小数点以下の桁数を指定するメソッドです。

    • 引数valに整数値を渡し、その数だけ小数点以下を表示します。set_digits(0)ならば、Gtk::SpinButtonに表示される数値は正数、set_digits(3)ならば、Gtk::SpinButtonに表示される数値は小数点以下３桁となります。
    • 表示桁数の制御として、set_digits()は表示する桁数を決めるもので、内部的な値の制度を制限するわけではありません。
    • 入力時の制約として、set_numeric(true)と組み合わせると、ユーザーは指定桁数までしか入力できません。
    • 丸め処理については、表示桁数を減らすと、見た目上は四捨五入されて表示されます。

  6. set_numeric( bool numeric )

    7. Gtk::SpinButtonに入力できる文字を数値のみに制限するためのメソッドです。

    • "true" : 数字と小数点以外の文字を入力できなくなります(デフォルト)。
    • "false" : 任意の文字列を入力可能になります（ただし数値以外を入力すると内部的に解釈できず、値は更新されません）。
    • set_digits(n)を設定している場合は、小数点以下ｎ桁まで入力が可能です。

  7. set_wrap(bool)

    8. Gtk::SpinButtonの値が、最大値や最小値の範囲を超えたときにどのように処理するかを制御するメソッドです。

    • "true" : 値が最大値を超えると 最小値に戻り、値が最小値を下回ると最大値に戻る、いわゆる「ループ」「循環」動作になります。
    • "false" : 範囲外に行かないように 値が固定されます(デフォルト)。

  8. シグナルの発生

    9. ボタンをクリックするか、数値を入力して確定したときにイベントシグナル(signal_value_changed)を発生させてシグナルハンドラーon_spin_changed()を起動します。



サンプル プログラム


#include <gtkmm.h>
#include <cmath>

class MyArea : public Gtk::DrawingArea {
public:
    MyArea();
    virtual ~MyArea() = default;
    void set_radius( double r );

private:
    double radius;
    void on_draw( const Cairo::RefPtr<Cairo::Context>& cr, int width, int height );
};

MyArea::MyArea()
 : radius( 50.0 )
{
    set_content_width( 200 );
    set_content_height( 200 );
    set_draw_func(sigc::mem_fun( *this, &MyArea::on_draw ));
}

void MyArea::set_radius( double r )
{
    radius = r;
    queue_draw();
}

void MyArea::on_draw( const Cairo::RefPtr<Cairo::Context>& cr, int width, int height )
{
    cr->set_source_rgb( 0.2, 0.6, 0.8 );
    cr->arc( width / 2.0, height / 2.0, radius, 0, 2 * M_PI );
    cr->fill();
}

class MyWindow : public Gtk::Window {
public:
    MyWindow();
    virtual ~MyWindow() = default;

private:
   // signal handler:
    void on_spin_changed();
   // child widget:
    Gtk::Box m_box;
    Gtk::SpinButton m_spin;
    MyArea m_area;
};

MyWindow::MyWindow()
{
    set_title( "SpinButton" );
    set_default_size( 300, 300 );

   // スピンボタンの範囲を設定
    m_spin.set_range( 10.0, 100.0 );   // 最小値10, 最大値100
    m_spin.set_increments( 0.5, 5.0 ); // ステップ1, ページステップ5
    m_spin.set_value( 50.0 );          // 初期値
    m_spin.set_digits( 1 );            // 少数第一位まで表示
    m_spin.set_numeric( true );        // 入力は数値のみ
    m_spin.set_wrap( true );           // 循環

   // スピンボタンの値変更時にDrawingAreaを更新
    m_spin.signal_value_changed().connect( sigc::mem_fun( *this, &MyWindow::on_spin_changed ));

    m_box.set_orientation( Gtk::Orientation::VERTICAL );
    m_box.set_margin( 10 );
    m_box.append( m_area );
    m_box.append( m_spin );

    set_child( m_box );
}

void MyWindow::on_spin_changed()
{
    m_area.set_radius( m_spin.get_value() );
}

int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "spinbutton.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
