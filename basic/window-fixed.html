Windowのサイズの固定


概 要

機　能

Windowのサイズを固定します。

使い方

  • set_resizable( false )でwindowのサイズを固定します。
  • また、set_resizable( true )でWindowのサイズを可変にすることができます。

サンプル プログラム

update:2024/11/8


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow( );
    virtual ~MyWindow() = default;
};
        
MyWindow::MyWindow( )
{
    set_title( "Windowのサイズの固定" );
    set_default_size( 320, 240 );
            
    set_resizable( false );    // Windowのサイズの固定
}
        
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>(argc, argv);
}
