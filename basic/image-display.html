画像ファイルの貼り付け

(update:2025/2/1)


概 要

機　能

画像ファイルをラベルやボタンに貼り付けて表示します。

使い方

  • 画像ファイルをラベルやボタンに貼り付けて表示します。
  • Gtk::Imageに画像データを読み込みます。
  • 次にGtk::BoxにGtk::ImageとGtk::Labelを追加します。
  • Gtk::Imageは、Gtk::Buttonの大きさにあわせて画像ファイルを拡大あるいは縮小します。

サンプル プログラム


#include <gtkmm.h>
#include <iostream.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;

protected:
   // Signal Handlers:
    void on_button_clicked();
  
   // Child widgets:
    Gtk::Button m_button;
    Gtk::Box    hbox;
    Gtk::Image  m_image;
    Gtk::Label  m_label;
};
  
MyWindow::MyWindow()
 : hbox( Gtk::Orientation::HORIZONTAL, 5 ),
   m_image( "balloon.png" ),
   m_label( "Balloon" )
{
    set_title( "Button" );
    m_image.set_expand( true );
    m_label.set_expand( true );

    hbox.append( m_image );
    hbox.append( m_label );
  
    m_button.set_child( hbox );
    m_button.signal_clicked().connect( sigc::mem_fun( *this,
            &MyWindow::on_button_clicked ));
    m_button.set_margin( 10 );
  
    set_child( m_button );
}
  
void MyWindow::on_button_clicked()
{
    std::cout << "The Button is clicked." << std::endl;
}
  
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
