CSS レイアウト

(update:2025/7/25)

概 要

機　能

ウイジットの基本的なレイアウトについてご紹介します。次図「CSS Box model」に示すように表示すべきラベルや図などのウイジットをcontentとして、contentの周囲にpadding、border、marginが配置されています。


使い方

margin
  1. 役割

    2. 要素の外側の余白

    marginプロパティは、は、ウイジットの外側の余白を指定するプロパティです。要素と他の要素との間に余白を設定し、要素間の間隔を調整するために使用されます。marginは、要素の外側にスペースを追加し、要素の表示を調整するために非常に重要です。

  2. 指定方法

    3. 値の指定

    marginの値は、ピクセル(px)、ポイント(pt)、em、ex、rem、パーセント(%)、またはautoで指定できます。autoを指定すると、ブラウザが自動的に余白を計算してくれます。

    値の一括指定

    marginは、まとめて指定することも可能です。例えば、`margin: 10px 20px 30px 40px;`と記述すると、上から時計回りに、それぞれ10px、20px、30px、40pxの余白を指定したことになります。

    値の個別指定

    marginは、ウイジットの上下左右それぞれに異なる余白を設定できます。例えば、margin-topで上側の余白、margin-bottomで下側の余白、margin-leftで左側の余白、margin-rightで右側の余白を指定できます。

    paddingとの違い

    marginはウイジットの外側の余白を指定するのに対し、paddingはウイジットの内側の余白を指定します。paddingはウイジットの背景色を内側の余白にも適用しますが、marginはウイジットの背景色を適用しません。

border
  1. 役割

    2. borderプロパティは、は、ウイジットの周りに枠線を追加するためのプロパティです。枠線の太さ、種類、色などが指定できます。ウイジットを強調したり、レイアウトを調整する際に使用します。

  2. 基本的な使い方

    3. 線の太さ(border-width)

    線の太さを指定します。 ( 例．8px、thin、medium、thick )

    線の種類(border-style)

    線の種類を指定します。 ( 例．solid(実践)、dashed(破線)、dotted(点線)、double(二重線) )

    線の色(border-color)

    線の色を指定します。 ( 例．Violet、#00ff00(Lime)、rgb(ff, ff, 00)(Yellow) )

    値の個別指定

    border-top, border-right, border-bottom, border-leftを用いて、上下左右それぞれの枠線を個別に指定します。

    線の角の丸め(border-radius)

    線の角を丸くします。 ( 例．border-radius: 50% 50%; 円 )

padding
  1. 役割

    2. paddingプロパティは、ウイジットの内側の余白を指定するために使用されます。ウイジットの内容と境界線（ボーダー）の間にスペースを作成し、ウイジットの見栄えを調整するために利用されます。

  2. 指定方法

    3. 値の指定

    余白の大きさは、ピクセル (px)、ポイント(pt)、em、ex、rem、パーセント (%)、またはautoで指定できます。autoを指定すると、ブラウザが自動的に余白を計算してくれます。

    値の一括指定

    padding: 上 右 下 左; のように、上下左右の余白をまとめて指定できます。例えば、padding: 10px 20px 10px 20px; は、上10px、右20px、下10px、左20px の余白を指定します。

    値の個別指定

    paddingは、コンテンツの上下左右それぞれに異なる余白を設定できます。例えば、padding-topで上側の余白、padding-bottomで下側の余白、padding-leftで左側の余白、paddingで右側の余白を指定できます。

    marginとの違い

    padding は要素の内側、margin は要素の外側の余白を指定します。margin は要素同士の間隔を調整するために使われます。


サンプル プログラム


#include <gtkmm.h>
#include <iostream>
#include <gtkmm/cssprovider.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;

private:
    Gtk::Label m_label1, m_label2, m_label3;

protected:
   // signal handler:
    static void on_parsing_error( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err );
   // child widgets:
    Gtk::Box m_box;
    Glib::RefPtr<Gtk::CssProvider> m_refCssProvider;
};

MyWindow::MyWindow()
 : m_box(  Gtk::Orientation::VERTICAL )
{
    set_title( "Css" );
    set_default_size( 320, 240 );
    set_child( m_box );
    m_box.set_spacing( 15 );
    m_box.set_name( "m-box" );

    m_label1.set_text( "Label 1" );
    m_label1.set_name( "m-label_A" );
    m_box.append( m_label1 );

    m_label2.set_text( "Label 2" );
    m_label2.set_name( "m-label_B" );
    m_box.append( m_label2 );

    m_label3.set_text( "Label 3" );
    m_label3.set_name( "m-label_C" );
    m_box.append( m_label3 );

   // Load extra CSS file
    m_refCssProvider = Gtk::CssProvider::create();   
    Gtk::StyleContext::add_provider_for_display( get_display(), m_refCssProvider,
        GTK_STYLE_PROVIDER_PRIORITY_APPLICATION );
    
    m_refCssProvider->signal_parsing_error().connect(
        []( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err )
        { on_parsing_error( sec, err ); }
    );   
    
    m_refCssProvider->load_from_path( "style.css" );
}

void MyWindow::on_parsing_error( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err )
{
    Gtk::CssLocation s_location, e_location;

    std::cerr << "on_parsing_error(): " << err.what() << std::endl;
    if ( sec ) {
        Glib::RefPtr<const Gio::File> file = sec->get_file();
        if ( file ) {
            std::cerr << "  URI = " << file->get_uri() << std::endl;
        }

        s_location = sec->get_start_location();
        e_location = sec->get_end_location();
        std::cerr << "  start : " << s_location.get_lines()+1
                  << ", end   : " << e_location.get_lines()+1 << std::endl;
        std::cerr << "  s_pos : " << s_location.get_line_chars()
                  << ", e_pos : " << e_location.get_line_chars() << std::endl;
    }
}

int main( int argc, char* argv[] )
{
    auto app = Gtk::Application::create( "gtkmm4.example" );   
    return app->make_window_and_run<MyWindow>( argc, argv );
}
