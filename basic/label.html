Labelの表示

(update:2025/2/1)

概 要

機　能

WidgetにLabelを表示します。

使い方

  • 作成したい label widget を Gtk::Window にset_child()で貼り付けます。
  • set_text( "Name" )でラベルの文字列をセットします。

プログラム


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
private:
    Gtk::Label m_label;
};
        
MyWindow::MyWindow()
{
    set_title("Label");
    set_default_size( 320, 240 );
        
    set_child( m_label );
    m_label.set_text( "Label example" );
}
        
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create("org.gtkmm4.examples");
    return app->make_window_and_run<MyWindow>(argc, argv);
}
