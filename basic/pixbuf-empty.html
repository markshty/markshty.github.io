空のpixbuf領域の作成

(update:2025/2/1)


概 要

機　能

Gdk::pixbufに空の画像領域を作成します。

使い方

  • Gdk::pixbuf( Gdk::Colorspace::RGB, false, 8, width, height )で幅width×高さheightの領域を作成します。
  • 第一引数は、Gdk::Colorspace::RGBは固定です。
  • 第二引数は、領域の画像データがアルファ値を持つならture、持たない場合はfalseを指定します。trueの場合は、fill()を実行したときにアルファ値が有効になります。
  • 第三引数は、８に固定です。
  • 作成された領域は、fill( color )によって着色することができます。
  • color: 例 0xffffff(white) 0x000000(black)

サンプル プログラム


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
  
private:
    Gtk::Image m_image;
};
  
MyWindow::MyWindow()
{
    set_title( "Background Color" );
    set_default_size( 640, 480 );
  
    Glib::RefPtr<Gdk::Pixbuf> pixbuf = Gdk::Pixbuf::create( Gdk::Colorspace::RGB, false, 8, 150, 125 );
    pixbuf->fill( 0x3355aaff ); 
    m_image.set( pixbuf ); 
  
    set_child( m_image );
}
  
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
