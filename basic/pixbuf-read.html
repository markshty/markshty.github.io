画像ファイルの読み込み


概 要

  • Gdk::pixbufは、様々なフォーマットの画像ファイルを読み込むことができます。
  • 読み込まれた画像データは、別のフォーマットで画像ファイルに書き出したり、ラベルやボタンに表示することができます。
  • また、画像データはグラフィック画面(Gtk::DrawingArea)に表示することもできます。
  • その他画像データを回転させたり、拡大・縮小させることができます。
  • 画像ファイルを読み込むためには、Gdk::Pixbuf::create_from_file("画像ファイル名")を使用します。

Sample Program

update:2024/11/8


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
  
private:
    Gtk::Image m_image;
    Glib::RefPtr<Gdk::Pixbuf> data;
};
  
MyWindow::MyWindow()
{
    set_title( "画像の読み込み" );
    set_default_size( 256, 256 );
  
   // 画像ファイルの読み込み
    data = Gdk::Pixbuf::create_from_file( "test.png" );
  
   // 画像ファイルを読み込んで表示
    m_image.set( data );
  
   // ウィンドウに画像を追加
    set_child( m_image );
}
  
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
