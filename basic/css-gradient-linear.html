CSS 線形グラデーション

(update:2025/7/25)

概 要

機　能

グラデーションは、2色以上の色が連続的に変化することでWindowの背景やウィジットに様々な色彩の効果を加えることができます。グラデーションの種類は線形グラデーション、放射形グラデーション、扇形グラデーション、反復形グラデーションの４つがあります。ここでは、まず、線形グラデーションについて記します。


使い方

  1. 基本形

    2. グラデーションでは、基本的に2色以上の色を指定することが必要です。これらは「color stops」と呼ばれており、日本語では「色経由点」あるいは「色の中間点」と訳すことがあります。デフォルトでは、上から下に向かって色が変化します。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
window {
    background-image: linear-gradient( blue, black);
}
    実行結果


  2. グラデーションの方向

    3. 線形グラデーションでは、デフォルトでは上から下に向かって色が変化します。色が変化する方向を上下左右に指定することができます。

    　 　 　 　 　 　 　 　 　　 　 　 　 　 　 　　 　 　 　
    書　式方　向 書　式方　向
    to top下から上へto right左から右へ
    to bottom上から下へto left右から左へ

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
window {
    background-image: linear-gradient( to right, blue, black );
}
    実行結果


  3. 対角線上のグラデーション

    4. 色が変化する方向を角から角に指定することができます。

    書　式 方　向 書　式 方　向
    to top right 右上角へ to bottom right 右下角へ
    to bottom left 左下角へ to top left 左上角へ

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
window {
    background-image: linear-gradient( to bottom right, blue, yellow );
}
    実行結果


  4. 角度の指定

    5. グラデーションに任意の角度を指定することができます。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
window {
    background-image: linear-gradient( 65deg, blue, yellow );
}
    実行結果


    角度を指定する場合、0degは下から上に向かうグラデーションとなります。90degは左から右へのグラデーションを生成し、プラスの角度を指定すると時計回りに回転し、マイナスの角度を指定すると反時計回りに回転します。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test2.cpp
    style.css
    
#m-label1 {
    font-size: 50px;
    background-image: linear-gradient( 0deg, blue, white );
}

#m-label2 {
    font-size: 50px;
    background-image: linear-gradient( 90deg, blue, white );
}

#m-label3 {
    font-size: 50px;
    background-image: linear-gradient( 180deg, blue, white );
}

#m-label4 {
    font-size: 50px;
    background-image: linear-gradient( -90deg, blue, white );
}
    実行結果


  5. 三色以上の使用

    6. CSSでは、三色以上の色を同時に使用することができます。デフォルトの設定では、各色は均等に配分されて表示されます。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
window {
    background-image: linear-gradient( blue, yellow, green,  orange );
}
    実行結果


  6. 色経由点(color stops)の配置

    7. color stopsの配置は、任意の位置を指定することができます。指定する値としては、パーセント値を使用するか、放射・線形グラデーションの場合は、具体的な長さを指定します。

    color stopsをパーセント値で指定する場合は、0%が開始点を表し、100％が終了点を表します。color stopsを指定しない場合は、その値はシステムによって自動的に計算され、最初のcolor stopsは 0% となり、最後のcolor stopsは、 100% となり、他のcolor stopsは、両側のcolor stopsの中間の位置となります。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
window {
    background-image: linear-gradient( to left, blue 25px, yellow 75%, green );
}
    実行結果


  7. 不連続線の生成

    8. 二つ以上のcolor stopsが同じ位置である場合、その位置で宣言された最初と最後の色の間で明確な線になります。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test1.cpp
    style.css
    
window {
    background-image: linear-gradient(
        to bottom right,
        SeaGreen 50%,
        orange   50%
    );
}
    実行結果


  8. 色の帯と縞の生成

    9. グラデーションの中に単色で遷移しない色の領域を含めるためには、異なる位置に同じ色を持つ二つの連続したcolor stopsをセットします。

    ｻﾝﾌﾟﾙ ﾌﾟﾛｸﾞﾗﾑ
    test3.cpp
    style.css
    
#m-label1 {
    background-image: linear-gradient(
        to top,
        SpringGreen 20%,
        red    30%,
        red    45%,
        Aqua   55%,
        Aqua   70%,
        orange 80%
    );
}

#m-label2 {
    background-image: linear-gradient(
        to top,
        SpringGreen 25%,
        red    25%,
        red    50%,
        Aqua   50%,
        Aqua   75%,
        orange 75%
    );
}
    実行結果



サンプル プログラム１


#include <gtkmm.h>
#include <iostream>
#include <gtkmm/cssprovider.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;

protected:
   // signal handler:
    static void on_parsing_error( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err );
   // child widgets:
    Glib::RefPtr<Gtk::CssProvider> m_refCssProvider;
};

MyWindow::MyWindow()
{
    set_title( "Css linear-gradient" );
    set_default_size( 320, 240 );
    
   // Load extra CSS file
    m_refCssProvider = Gtk::CssProvider::create();   
    Gtk::StyleContext::add_provider_for_display( get_display(), m_refCssProvider,
        GTK_STYLE_PROVIDER_PRIORITY_APPLICATION );
    m_refCssProvider->signal_parsing_error().connect(
        []( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err )
        { on_parsing_error( sec, err ); }
    );   
    m_refCssProvider->load_from_path( "style.css" );
}

void MyWindow::on_parsing_error( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err )
{
    Gtk::CssLocation s_location, e_location;

    std::cerr << "on_parsing_error(): " << err.what() << std::endl;
    if ( sec ) {
        Glib::RefPtr<const Gio::File> file = sec->get_file();
        if ( file ) {
            std::cerr << "  URI = " << file->get_uri() << std::endl;
        }

        s_location = sec->get_start_location();
        e_location = sec->get_end_location();
        std::cerr << "  start : " << s_location.get_lines()+1
                  << ", end   : " << e_location.get_lines()+1 << std::endl;
        std::cerr << "  s_pos : " << s_location.get_line_chars()
                  << ", e_pos : " << e_location.get_line_chars() << std::endl;
    }
}

int main( int argc, char* argv[] )
{
    auto app = Gtk::Application::create( "gtkmm4.example" );   
    return app->make_window_and_run<MyWindow>( argc, argv );
}

#include <gtkmm.h>
#include <iostream>
#include <gtkmm/cssprovider.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;

private:
    Gtk::Box   m_box;
    Gtk::Label m_label1, m_label2, m_label3, m_label4;

protected:
   // signal handler:
    static void on_parsing_error( const Glib::RefPtr<const Gtk::CssSection>& sec, const Glib::Error& err );
   // child widgets:
    Glib::RefPtr<Gtk::CssProvider> m_refCssProvider;
};

MyWindow::MyWindow()
 : m_box( Gtk::Orientation::VERTICAL )
{
    set_title( "Css linear-gradient" );
    set_child( m_box );
    m_box.set_margin( 20 );
    m_box.set_spacing( 20 );

    m_label1.set_size_request( 150, 150 );
    m_label1.set_text( "0度" );
    m_label1.set_name( "m-label1" );
    m_box.append( m_label1 );

    m_label2.set_size_request( 150, 150 );
    m_label2.set_text( "90度" );
    m_label2.set_name( "m-label2" );
    m_box.append( m_label2 );

    m_label3.set_size_request( 150, 150 );
    m_label3.set_text( "180度" );
    m_label3.set_name( "m-label3" );
    m_box.append( m_label3 );

    m_label4.set_size_request( 150, 150 );
    m_label4.set_text( "-90度" );
    m_label4.set_name( "m-label4" );
