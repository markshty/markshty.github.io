Windowの表示


概 要

機　能

Windowを表示します。

使い方

  • 作成したい Window widget を Gtk::Window から継承します
  • set_title()でwindowのタイトルをセットすることができます。
  • set_default_size( int width, int height )でWindowのサイズを指定することができます。
  • widthはwindowの幅、heightはwindowの高さ
  • Gtk::Application::create()でプログラムのタイトルをセット。
>

サンプル プログラム

update:2024/11/8


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
};
        
MyWindow::MyWindow()
{
    set_title( "Basic application" );
    set_default_size( 320, 240 );
}
        
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.examples" );
    return app->make_window_and_run<MyWindow>(argc, argv);
}
