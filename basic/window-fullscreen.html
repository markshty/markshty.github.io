Windowの最大化(FullScreen)

(update:2025/2/1)


概 要

機　能

Windowのサイズを画面に合わせて最大化します。

使い方

  • fullscreen()でwindowを画面に合わせて最大化します。
  • また、unfullscreenでWindowのサイズを元のサイズにもどします。

サンプル プログラム


#include <gtkmm/application.h>
#include <gtkmm/box.h>
#include <gtkmm/button.h>
#include <gtkmm/window.h>
#include <iostream>
      
class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
      
protected:
   // Signal handlers:
    void on_button1_clicked();
    void on_button2_clicked();
      
   // Child widgets:
    Gtk::Box m_box;
    Gtk::Button m_button1, m_button2;
};      
      
MyWindow::MyWindow()
 : m_box( Gtk::Orientation::VERTICAL, 10 ),
   m_button1( "on" ),
   m_button2( "off" )
{
   // This just sets the title of our new window.
    set_title( "Full Screen" );
      
   // Sets the margin around the box.
    m_box.set_margin( 10 );
      
   // put the box into the main window.
    set_child( m_box );
      
    m_button1.signal_clicked().connect(
       sigc::mem_fun(*this, &MyWindow::on_button1_clicked));
    m_box.append( m_button1 );
    m_button1.set_expand();
      
    m_button2.signal_clicked().connect(
       sigc::mem_fun(*this, &MyWindow::on_button2_clicked));
    m_box.append( m_button2 );
    m_button2.set_expand();
}
      
void MyWindow::on_button1_clicked()
{
    fullscreen();    // 最大化
    std::cout << "ON" << std::endl;
}
      
void MyWindow::on_button2_clicked()
{
    unfullscreen();    // 元のサイズ
    std::cout << "OFF" << std::endl;
}
      
int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.example" );
    return app->make_window_and_run<MyWindow>(argc, argv);
}

実行結果

Button[On]をクリックするとWindowが画面一杯に拡大されます。

Button[Off]をクリックするとWindowが元のサイズにもどります。
