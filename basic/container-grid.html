Gtk::Grid

(update:2025/7/25)

概 要

機　能

Gtk::Gridは、multi-item containerです。Gtk::Boxの中に複数のウィジットを格納することができます。


レイアウト

Gtk::Gridのレイアウトを次に示します。行列上に子ウィジットを配置します。子ウィジットの行間の間隔と列間の間隔を指定することができます。

使い方

  1. Gtk::Gridの宣言

    2. Gtk::Boxのwidgetの宣言を行います。


  2. マージンの指定

    3. Gtk::Gridの周囲にマージンを指定します。指定する数値はpixel単位です。set_margin()では、4辺の周囲に同じマージンをセットします。

    【関　数】

    void Gtk::Widget::set_margin( int margin )

    【返　値】 : 無し

    ※マージンを指定する方法は、この他に各辺ごとにマージンをセットすることができます。
    set_margin_top():上端、set_margin_start():左端、set_margin_end():右端、set_margin_bottom():下端


  3. 行間の指定

    4. Gtk::Gridが格納する子ウィジットの行間の間隔を指定します。指定する数値はpixel単位です。

    【関　数】

    void Gtk::Grid::set_row_spacing( guint spacing )

    【返　値】 : 無し


  4. 列間の指定

    5. Gtk::Gridが格納する子ウィジットの列間の間隔を指定します。指定する数値はpixel単位です。

    【関　数】

    void Gtk::Grid::set_column_spacing( guint spacing )

    【返　値】 : 無し


  5. 子ウィジットの追加

    6. Gtk::Gridに子ウィジットを追加します。


  6. Gtk::Gridを親ウィジットの子ウィジットとして指定

    7. Gtk::Gridを子Widgetとして親Widgetにセットします。

    【関　数】

    void Gtk::Widget::append( Gtk::Grid grid )

    【返　値】 : 無し


  7. Gtk::Grid内の子ウィジットの配置A

    8. Gtk::Gridの子Widgetの配置を親Widgetにセットします。

    【関　数】

    void Gtk::Grid:attach( Widget & child, // 子ウィジット
    int column, // 子ウィジットを配置する列
    int row, // 子ウィジットを配置する行
    int width=1, // 子ウィジットを配置する行数
    int height=1 ) // 子ウィジットを配置する列数

    【返　値】 : 無し


  8. Gtk::Grid内の子ウィジットの配置B

    9. Gtk::Gridの子Widgetの配置を親Widgetにセットします。

    【関　数】

    void Gtk::Grid:attach_next_to( Widget & child, // 子ウィジット
    PositionType side, // 子ウィジットを配置する位置
    int width=1, // 子ウィジットを配置する行数
    int height=1 ) // 子ウィジットを配置する列数

    【返　値】 : 無し


    Gtk::PositionType
    書式 内容
    LEFT 子ウィジットを左端に配置
    RIGHT 子ウィジットを右端に配置
    TOP 子ウィジットを上端に配置
    BOTTOM 子ウィジットを下端に配置


  9. Gtk::Grid内の子ウィジットの行の幅を揃える

    10. Gtk::Gridに格納する子ウィジットの行の幅を均等に揃えます。

    【関　数】

    void Gtk::GridGrid::set_row_homogeneous( bool homogeneous=true )

    【返　値】 : 無し


  10. Gtk::Grid内の子ウィジットの列の幅を揃える

    11. Gtk::Gridに格納する子ウィジットの列の幅を均等に揃えます。

    【関　数】

    void Gtk::GridGrid::set_column_homogeneous( bool homogeneous=true )

    【返　値】 : 無し


サンプル プログラム


#include <gtkmm.h>
#include <iostream>

class MyWindow : public Gtk::Window {
public:
    MyWindow();
    virtual ~MyWindow() = default;
protected:
   // Signal handlers:
    void on_button_clicked( const Glib::ustring& data );
    void on_button_quit();
    void on_button_test();
    
   // Child widgets:
    Gtk::Button m_button_test, m_button_quit;
    Gtk::Box m_box;
   // 1.宣言
    Gtk::Grid m_grid;
};

MyWindow::MyWindow()
 : m_box( Gtk::Orientation::VERTICAL ),
   m_button_test( "Test" ),
   m_button_quit( "Close" )
{
    set_title( "Box" );
    set_child( m_box );
    m_box.set_margin( 5 );
    m_box.set_spacing( 10 );
   // 2.マージンの指定
    m_grid.set_margin( 10 );
   // 3.行間の指定
    m_grid.set_row_spacing( 15 );
   // 4.列間の指定
    m_grid.set_column_spacing( 5 );
   // 5.親ウィジットの子ウィジットとして指定
    m_box.append( m_grid );

    for( int i = 0; i < 3; i++ ) {
        for( int j = 0; j < 3; j++ ) {
            auto d = Glib::ustring::compose( "%1", i * 3 + j + 1 );
            auto pbutton = Gtk::make_managed<Gtk::Button>( d );
            pbutton->signal_clicked().connect( sigc::bind(
                sigc::mem_fun( *this, &MyWindow::on_button_clicked), std::to_string( i * 3 + j + 1 )));
           // 6.子ウィジットの配置A
            m_grid.attach( *pbutton, i, j );
        }
    }

    m_button_test.signal_clicked().connect( sigc::mem_fun( *this, &MyWindow::on_button_test ));
   // 7.子ウィジットの配置B
    m_grid.attach_next_to( m_button_test, Gtk::PositionType::BOTTOM, 3, 1 );

    m_button_quit.signal_clicked().connect( sigc::mem_fun( *this, &MyWindow::on_button_quit ));
    m_button_quit.set_halign( Gtk::Align::END );
    m_box.append( m_button_quit );
}

void MyWindow::on_button_clicked( const Glib::ustring& data )
{
    std::cout << "output : " << data << std::endl;
}

void MyWindow::on_button_quit()
{
    set_visible( false );
}

void MyWindow::on_button_test()
{
    std::cout << "test" << std::endl;
}

int main(int argc, char *argv[]) 
{
    auto app = Gtk::Application::create( "gtkmm4.examples" );
    return app->make_window_and_run<MyWindow>(argc, argv);
}
