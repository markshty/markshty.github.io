画像の読み込み


概 要

機　能

Gtk::Imageに画像データを読み込みます。

使い方

  • set( 画像ファイル )で画像データをGtk::Imageに読み込みます。
  • png bmp gif xpm svg の画像データを読み込むことができます。

サンプル プログラム

update:2024/11/8


#include <gtkmm.h>

class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;
  
private:
    Gtk::Image m_image;
};
  
MyWindow::MyWindow()
{
    set_title( "画像の読み込み" );
    set_default_size( 256, 256 );
  
   // 画像ファイルを読み込んで表示
    m_image.set("./test.png");
  
   // ウィンドウに画像を追加
    set_child(m_image);
}

int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "gtkmm4.examples" );
    return app->make_window_and_run<MyWindow>(argc, argv);
}
