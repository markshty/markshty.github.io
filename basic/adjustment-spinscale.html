Adjustment(SpinButtonやScaleとの連携)

(update:2026/1/2)

概 要

目　的

SpinButtonやScaleとの連携の例として、Gtk::SpinButton と Gtk::Scale に同じ Gtk::Adjustment を共有させ、値を同期させるプログラムを作成します。

構　成

プログラムの構成は次の通りです。

  1. Gtk::Adjustmentの参照ポインタの宣言
    • Gtk::Windowの派生クラスとして、MyWindowクラスを宣言します。
    • MyWindowクラスの中で変数として、Gtk::ScaleGtk::SpinButtonを宣言します。
    • Gtk::Adjustmentの参照ポインタを宣言します。

  2. Gtk::Adjustmentの生成

    3. Gtk::Adjustment::create（）を用いて、Gtk::Adjustmentの実体を生成します。


  3. Gtk::Adjustmentと他のwidgetの連携
    • set_adjustment()を用いて、Gtk::AdjustmentとGtk::SpinButtonを結びつけます。
    • 同様にset_adjustment()を用いて、Gtk::AdjustmentとGtk::Scaleを結びつけます。

  4. イベントシグナルの設定

    5. スピンボタンやスライダーを操作したときにGtk::adjustmentの(signal_value_changed)を発生させてシグナルハンドラーon_value_changed()を起動します。


  5. シグナルハンドラーの起動
    • Gtk::Adjustmentの値(value)の取得

      • スピンボタンやスライダーが操作されたときに、値(value)をget_value()を実行して取得し、set_radius()に引き渡します。

    • Gtk::DrawingAreaの再描画の実行

      • set_radius()の中でGtk::Drawingareaの再描画を実行します。


サンプル プログラム


#include <gtkmm.h>
#include <cmath>

class MyArea : public Gtk::DrawingArea
{
public:
    MyArea();
    virtual ~MyArea() = default;
    void set_radius( double r );

private:
   // signal handler:
    void on_draw(const Cairo::RefPtr<Cairo::Context>& cr, int width, int height);
   // child widget:
    double radius_;
};

MyArea::MyArea()
 : radius_( 50.0 )
{
    set_content_width( 200 );
    set_content_height( 200 );
    set_draw_func( sigc::mem_fun( *this, &MyArea::on_draw ));
}

void MyArea::set_radius( double r )
{
    radius_ = r;
    queue_draw(); // 5.2 Gtk::DrawingAreaの再描画の実行
}

void MyArea::on_draw( const Cairo::RefPtr<Cairo::Context>& cr, int width, int height )
{
    cr->set_source_rgb( 0.4, 0.4, 0.9 );
    cr->arc( width / 2.0, height / 2.0, radius_, 0, 2 * M_PI );
    cr->fill();
}


class MyWindow : public Gtk::Window
{
public:
    MyWindow();
    virtual ~MyWindow() = default;

private:
   // signal_handler:
    void on_value_changed();

   // child widget:
    Gtk::Box        m_box;
    Gtk::SpinButton m_spin;
    Gtk::Scale      m_scale;
    MyArea          m_area;
    // 1.Gtk::Adjustmentの参照ポインタの宣言
    Glib::RefPtr<Gtk::Adjustment> m_adjustment;
    
};

MyWindow::MyWindow()
{
    set_title( "Adjustment : SpinButton & Scale");
    set_default_size( 300, 300 );

    // 2. Gtk::Adjustmentの生成
    m_adjustment = Gtk::Adjustment::create( 55.0, 10.0, 100.0, 1.0, 5.0 );

    // 3.1 Gtk::AdjustmentとGtk::SpinButtonの連携
    m_spin.set_adjustment( m_adjustment );
    m_spin.set_digits( 0 );
    m_spin.set_numeric( true );

    // 3.2 Gtk::AdjustmentとGtk::SpinButtonの連携
    m_scale.set_adjustment( m_adjustment );
    m_scale.set_digits( 0 );
    m_scale.set_draw_value( true );
    m_scale.set_orientation( Gtk::Orientation::HORIZONTAL );

    // 4. イベントシグナルの設定
    m_adjustment->signal_value_changed().connect(
        sigc::mem_fun( *this, &MyWindow::on_value_changed ));

    m_box.set_orientation( Gtk::Orientation::VERTICAL );
    m_box.set_margin( 10 );
    m_box.append( m_area );
    m_box.append( m_spin );
    m_box.append( m_scale );

    set_child( m_box );
}

// 5.シグナルハンドラーの起動
void MyWindow::on_value_changed()
{
    m_area.set_radius( m_adjustment->get_value() ); // 5.1 値(value)の取得
}

int main(int argc, char* argv[])
{
    auto app = Gtk::Application::create( "adjustment.example" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
