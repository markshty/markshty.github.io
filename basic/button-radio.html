RadioButton

(update:2025/12/14)

概 要

機　能

Gtkmm4では、Gtk::RadioButtonは削除されています。しかし、Gtk::CheckButtonやGtk::TogglrButtonをset_groupを使って、グループ化することにより、従前のRadioButtonと同様の機能を実現することができます。


使い方

Gtk::CheckButtonウイジットの使い方は次のとおりです。
  1. ボタンの作成

    2. Gtk::CheckButtonウイジットの作成は、Gtk::CheckButtonのコンストラクタでラベルとなる文字列を指定します。あるいは、Gtk::CheckButtonのコンストラクタで定義しておいて、ラベルの文字列は、set_label( "文字列" ) を使って後から指定します。


  2. ボタンのグループ化

    3. checkbutton2をcheckbutton1にグループ化する場合は、次の形になります。

    checkbutton2.set_group( checkbutton1 )


  3. シグナルの発生

    4. ボタンをクリックしたときにイベントシグナル(signal_toggled)を発生させてシグナルハンドラーon_button_toggled()を起動します。


  4. get_active()

    5. 現在の状態を取得（ON/OFF）。


  5. set_active( bool )

    6. プログラム上でボタンのON(true)、OFF(false)を指定する。



サンプル プログラム


#include <gtkmm.h>

class MyWindow : public Gtk::Window {
public:
    MyWindow();
    virtual ~MyWindow() = default;

private:
   // signal handler:
    void on_button_toggled();
   // child widget:
    Gtk::Box *m_box;
    Gtk::CheckButton* check1;
    Gtk::CheckButton* check2;
    Gtk::CheckButton* check3;
    Gtk::Label *status_label;
};

MyWindow::MyWindow()
{
    set_title( "RadioButton" );
    set_default_size( 300, 250 );

   // 内側のBox（ボタンとラベルを縦に並べる）
    m_box = Gtk::make_managed<Gtk::Box>( Gtk::Orientation::VERTICAL, 10 );

   // 中央寄せ指定（上下左右とも中央）
    m_box->set_valign( Gtk::Align::CENTER );
    m_box->set_halign( Gtk::Align::CENTER );

   // チェックボタンを3つ作成
    check1 = Gtk::make_managed<Gtk::CheckButton>( "Button 1");
    check2 = Gtk::make_managed<Gtk::CheckButton>( "Button 2" );
    check3 = Gtk::make_managed<Gtk::CheckButton>( "Button 3" );

   // グループ化
    // checkbutton2をcheckbutton1のグループに加える。
    check2->set_group( *check1 );
    // checkbutton3をcheckbutton1のグループに加える。
    check3->set_group( *check1 );

   // デフォルトで1つ選択
    check1->set_active( true );

   // ラベルを作成
    status_label = Gtk::make_managed<Gtk::Label>( "Selected: Button 1" );

   // シグナル接続
    check1->signal_toggled().connect(
        sigc::mem_fun( *this, &MyWindow::on_button_toggled ));
    check2->signal_toggled().connect(
        sigc::mem_fun( *this, &MyWindow::on_button_toggled ));
    check3->signal_toggled().connect(
        sigc::mem_fun( *this, &MyWindow::on_button_toggled ));

   // 内側Boxに追加
    m_box->append( *check1 );
    m_box->append( *check2 );
    m_box->append( *check3 );
    m_box->append( *status_label );

   // ウィンドウに直接 inner_box をセット
    set_child( *m_box );
}

void MyWindow::on_button_toggled()
{
    if (check1->get_active()) {
        status_label->set_text( "Selected: " + check1->get_label() );
    } else if (check2->get_active()) {
        status_label->set_text( "Selected: " + check2->get_label() );
    } else if (check3->get_active()) {
        status_label->set_text( "Selected: " + check3->get_label() );
    }
}

int main( int argc, char* argv[] )
{
    auto app = Gtk::Application::create( "radiobutton.examples" );
    return app->make_window_and_run<MyWindow>( argc, argv );
}
