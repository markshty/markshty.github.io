pdfファイル(PdfSurface)

(update:2025/3/8)

概 要

機　能

　PdfSurfaceを使用して図形や画像データをpdfファイルに書き込むことができます。ただし、PdfSurfaceを画面に表示することはできません。

使い方

  1. PdfSurfaceの領域の確保

    2. create()関数を使って、PdfSurfaceの領域を確保します。

    static RefPtr<PdfSurface>Cairo::PdfSurface::create( std::string filename, // ｻｰﾌｪｰｽ保存時のファイル名
    int width, // pdfファイルの横幅(単位:point)
    int height ) // pdfファイルの高さ(単位:point)

  2. Cairo::Contextの作成

    3. 領域確保したPdfSurfaceに対応するCairo::Contextを作成します。


  3. 線画の描画

    4. 上記のCairo::Contextを使って線画を描画します(パスを形成)。


  4. デフォルトのソースへの転送

    5. show_page()関数を実行すると、パスがPdfSurfaceを写しこまれます。


サンプル プログラム


#include <gtkmm.h>

int main(){
   // 1.PdfSurfaceの領域の確保
    Cairo::RefPtr<Cairo::PdfSurface> surface = Cairo::PdfSurface::create( "test.pdf", 320, 240 );
   // 2.Cairo::Contextの作成
    Cairo::RefPtr<Cairo::Context> crx = Cairo::Context::create( surface );
   // 3.線画の描画
    crx->set_source_rgba( 1.0, 1.0, 1.0, 1.0 );
    crx->paint();
  
    crx->set_source_rgba( 0.2, 0.5, 1.0, 1.0);
    crx->set_line_width( 0.5 );
    crx->move_to(   0, 120 );
    crx->line_to( 320, 120 );
    crx->stroke();
  
    crx->set_source_rgba( 0.1, 0.7, 0.8, 1.0);
    crx->move_to( 0, 240 );
    crx->rel_line_to( 320, 0);
    crx->rel_line_to( -120, -120 );
    crx->rel_line_to( -80, 0 );
    crx->close_path();
    crx->fill();
  
    crx->set_source_rgba( 1.0, 1.0, 0.0, 1.0 );
    crx->arc( 160, 60, 20, 0, 2*M_PI );
    crx->fill();

   // 4.画像の出力
    crx->show_page();
}
