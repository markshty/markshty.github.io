演算の実行時間の測定

(update:2026/1/2)

概 要

機　能

Glib::Timerは、Glibライブラリに含まれる軽量なタイマー機能で、経過時間の測定に使用することができます。ストップウォッチとして使用したり、関数等の演算の実行時間の測定に用いることができます。


使い方

関数等の演算の実行時間を測定する手順は、次の通りです。
  1. タイマーのwidgetの宣言

    2. Glib::Timerのwidgetを宣言します。

  2. 測定開始

    3. タイマーをスタートさせます。

  3. 演算

    4. ここで測定対象となる関数等を実行します。

  4. 測定終了

    5. stop()を実行してタイマーを停止させます。

  5. 測定結果の表示

    6. elapsed()を実行して経過時間を取得し、標準出力に表示します。


サンプル プログラム

ここでは、ｻﾝﾌﾟﾙﾌﾟﾛｸﾞﾗﾑとして、乱数生成の実行時間の測定を行います。ﾒﾙｾﾝﾇ・ﾂｲｽﾀ法を用いて、0.0～1.0の範囲の値を持つ実数型の乱数を１００万回生成します。 

#include <glibmm/timer.h>
#include <random>
#include <iostream>

int main() {
    Glib::Timer timer;        // １タイマーのwidgetの宣言
    std::random_device rnd;
    std::mt19937 mt( rnd() );
    std::uniform_real_distribution<> rand_B( 0.0, 1.0 );

   // ２測定開始
    timer.start(); 

   // ３演算
    for ( int i = 0; i < 1000000; i++ ) {
        rand_B( mt );
    }

   // ４測定終了
    timer.stop();  

   // ５測定結果の表示
    std::cout << "Elapsed time: " << timer.elapsed() << " seconds" << std::endl;
    return 0;
}
